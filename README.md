## 📝 Deskripsi
*SIGMA System* adalah aplikasi berbasis CLI (*Command Line Interface*) yang dikembangkan untuk membantu otomatisasi perhitungan nilai akademik mahasiswa. Program ini menghitung rata-rata dari 5 mata kuliah utama dan memberikan kategori nilai secara akurat melalui algoritma evaluasi yang presisi.

## 💡 Filosofi Nama (SIGMA)
Nama *SIGMA* merupakan manifestasi dari alur logika sistem ini:
* *S* ystem: Instruksi yang berjalan secara sistematis.
* *I* nformation: Mengubah data mentah menjadi informasi nilai.
* *G* rade: Fokus pada penentuan tingkatan prestasi (A-E).
* *M* ahasiswa: Berorientasi pada data akademik mahasiswa.
* *A* ccurate: Menghasilkan output perhitungan yang tepat.
